How to use letter box

Step 1. Create account on letter box

Step 2. Make request to https://letter-box.heroku.app/postletter/yourusername for posting letter.here is javascript code emplementation

      
        var myHeaders = new Headers();
        myHeaders.append("Content-Type", "application/json");

        var raw = JSON.stringify({
            "name" : name, 
            "email" : email,
            "message" : message,
        });

        var requestOptions = {
        method: 'POST',
        headers: myHeaders,
        body: raw,
        redirect: 'follow'
        };

        fetch(`${endpoint}/postletter/yourusername`, requestOptions)
        .then(response => response.text())
        .then(result => {
            console.log(result);
        })
        .catch(error => {
            console.log("error",error);
        });

Step 3. Login to letter box to see your messages.
